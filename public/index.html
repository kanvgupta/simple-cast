What is ChainView?

ChainView makes it easy to share Foundry Cast commands with others. Create, organize, and share blockchain interactions with a simple URL - no account required.

Try it with our CLI

$ npm install -g chainview
$ chainview submit "cast call --rpc-url=https://eth-mainnet.alchemyapi.io/v2/KEY TOKEN_ADDRESS 'balanceOf(address)' WALLET_ADDRESS" --title "Check ERC20 Balance"

You'll get back a shareable link that looks like:

https://chainview.kanvgupta.com/api/c/J1IePzJ75b

Features

Share Commands

Create and share Cast commands with a simple URL. Perfect for tutorials, documentation, or collaboration.

Organize Collections

Group related commands into collections. Create an ERC20 collection with common function calls.

CLI Tool

Use our simple CLI tool to create and manage commands and collections.

Getting Started

1

Install the CLI

npm install -g chainview
2

Create a Command

chainview submit "cast call --rpc-url=https://eth-mainnet.alchemyapi.io/v2/KEY 0xTOKEN 'function()' ARG" --title "My Command"
3

Create a Collection

chainview collection create "My Collection" --desc "Collection description"
4

Add Command to Collection

chainview collection add COLLECTION_ID "cast call ..." --title "New Command"
